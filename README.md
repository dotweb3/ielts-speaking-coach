# IELTS Speaking Coach

一个本地网页陪练器，围绕 3 秒开口、Part 2 稳定结构、Part 3 抽象展开和完整模拟。

## 打开方式

最简单的方式：双击这个文件并保持弹出的窗口不要关闭。

```text
start_ielts_coach.command
```

也可以在这个文件夹手动启动本地网页服务：

```bash
python3 server.py
```

本机打开：

```text
http://localhost:8080
```

同一局域网设备打开启动时打印的 `LAN` 地址，例如：

```text
http://192.168.x.x:8080
```

浏览器录音通常要求安全环境。本机 `localhost` 基本可用；手机或其他设备通过局域网 `http` 打开时，如果浏览器拦截麦克风，可以继续用计时训练，或改在本机打开录音。

## 麦克风说明

网页录音不是普通网页权限，它受浏览器安全策略限制：

- 电脑录音：请打开 `http://localhost:8080`，再点页面里的“启用麦克风”。
- 手机访问电脑网页：`http://192.168.x.x:8080` 通常不能调用麦克风，所以不会弹授权。
- 如果电脑用 `192.168.x.x:8080` 打开，也可能不会弹授权；录音请改用 `localhost`。
- 如果已经拒绝过权限，请在浏览器地址栏左侧的网站设置里重新允许麦克风，然后刷新页面。

## 功能

- 3 秒开口、Part 2、Part 3、完整模拟四种训练
- 考官读题、考场模式、倒计时和中途节奏提醒
- 麦克风录音、音量条、回放和录音下载
- 快速复盘标签、评分、历史记录、导入和导出
- 今日训练清单会根据练习数量自动点亮
